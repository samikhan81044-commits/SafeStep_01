# SafeStep – Personal Emergency Protection Application

## Overview

SafeStep is an Android-based personal emergency protection application designed to provide quick assistance during emergency situations. The application allows users to trigger SOS alerts using multiple activation methods, including Shake Detection, Voice Keyword Detection, and Hand Gesture Recognition.

The primary goal of SafeStep is to improve personal safety by enabling users to send emergency alerts and location information to trusted contacts without requiring complex manual actions.

---

## Problem Statement

In emergency situations, users may not have enough time to unlock their phones, open applications, or manually contact someone for help. Traditional emergency solutions often depend on a single SOS button, which may not be practical in critical situations.

SafeStep addresses this problem by providing multiple emergency activation methods that can quickly notify emergency contacts and share the user's location.

---

## Key Features

### 1. Shake Detection

Users can trigger emergency alerts by shaking their phone multiple times.

### 2. Voice Keyword Detection

The application continuously listens for predefined emergency keywords such as "Help" and triggers an SOS alert when detected.

### 3. Hand Gesture Detection

Using the device camera and hand recognition technology, users can activate emergency protection by showing an open palm gesture.

### 4. Emergency SOS Alerts

Automatically sends emergency SMS messages to saved emergency contacts.

### 5. Location Sharing

Shares the user's current GPS location along with emergency alerts.

### 6. Emergency Contact Management

Users can add, edit, and manage trusted emergency contacts.

### 7. Protection Control

Users can enable or disable protection services directly from the application dashboard.

### 8. Settings Customization

Allows users to customize:

* SOS Message
* Voice Keywords
* Shake Sensitivity
* Permission Status Monitoring

## Application Workflow

1. User opens SafeStep.
2. Required permissions are granted.
3. Emergency contacts are added.
4. Protection is enabled.
5. Emergency is detected through:

   * Shake Detection
   * Voice Keyword Detection
   * Hand Gesture Detection
6. SOS alert is triggered.
7. Emergency SMS and location are sent to trusted contacts.
